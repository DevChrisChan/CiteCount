Before You Add a Tool

A few constraints come from how iframes work on the web, not from CiteCount itself.

Use iframe-friendly URLs

Some websites block embedding with security headers. Internal CiteCount pages and purpose-built embeddable pages work best.

Keep metadata short

The More Tools grid is compact. Clear names and one-sentence descriptions make the workspace easier to scan and pin.

Add Custom Tool

Bring your own iframe-powered page into CiteCount. Relative paths like /support/ work for internal tools.
